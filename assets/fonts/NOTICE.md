# Bundled fonts

This folder ships **DejaVu Sans** (`DejaVuSans.ttf`, `DejaVuSans-Bold.ttf`) as the default rendering font, so text rendering works out of the box on any system without hunting for a system font.

DejaVu fonts are based on the Bitstream Vera Fonts and are distributed under a free license that permits use, redistribution, and modification. See the DejaVu Fonts license:

https://dejavu-fonts.github.io/License.html

You can use any other `.ttf` or `.otf` font by setting `"font": "path/to/font.ttf"` in your config.
